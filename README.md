
## Features
Task Management
Create, edit, and delete tasks
Organize tasks into To Do, Doing, and Done columns
Drag and drop tasks between columns
Reorder tasks within the same column
Quick actions for moving tasks without dragging
Task Details

## Each task supports:
Title
Description
Priority level (High, Medium, Low)
Due date
Status column
Search and Filtering
Search tasks by title or description
Filter tasks by priority
Filter tasks by workflow stage
Data Persistence
Automatic LocalStorage saving
Data remains available after page refreshes
Import and export task data as JSON
Dashboard
Total task count
High-priority task count
Tasks in progress
Completed tasks
Overdue tasks
Completion progress tracking
User Experience
Responsive design for desktop, tablet, and mobile devices
Dark and light theme support
Toast notifications for user actions
Undo recently deleted tasks
Keyboard shortcuts for common actions


## Keyboard Shortcuts
Shortcut	Action
N	Create new task
/	Focus search
Ctrl + K	Focus search
T	Toggle theme
Ctrl + E	Export tasks
Ctrl + Z	Undo delete
?	View shortcuts
Esc	Close modal
